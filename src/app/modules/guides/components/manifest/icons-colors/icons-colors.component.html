Иконки и цвета браузера.

Современные браузеры облегчают настройку определенных компонентов, таких как иконки, цвета адресной строки и даже пользовательских плиток. Эти простые настройки могут повысить впечатление от вашего сайта и повлиять на возможность повторного посещения его пользователями.

Представление иконок или плиток.

Когда пользователь посещает ваш сайт, браузер пытается получить изображения из HTML. Они могут появлятся во многих местах, включая закладки браузеров, недавние переходы к приложениям, и т.д. Предоставляя высококачественные изображения (иконки) делают ваш сайт наиболее узнаваемым, облегчая пользователям поиск вашего сайта. Для добавления иконок высокого разрешения с полной поддержкой для всех браузеров, необходимо добавить несколько тегов link / meta в область тега head на каждой станице.

<!-- нужная иконка высокого разрешения --> <link rel="icon" sizes="192x192" href="icon.png"> <!-- повторно используемая иконка в Safari --> <link rel="apple-touch-icon" href="ios-icon.png"> <!-- несколько иконок для IE --> <meta name="msapplication-square310x310logo" content="icon_largetile.png">
Chrome и Opera.

Этибраузеры используют изображения расширения .png, которые масштабируются в зависимости от размеров устройств. Для предотвращения автомасштабирования необходимо использовать атрибут sizes, добавляя дополнительные размеры. Размеры изображений должны иметь основание 48px (24px, 48px, 96px, 144px, 192px).

Safari.

Этот браузер также использует элемент link с атрибутом apple-touch-icon, для определение изображений. Так же, можно устанавливать явные размеры, определяя отдельные элементы для каждой иконки, предотвращая автоматическое масштабирование операционной системой.

<link rel="apple-touch-icon" href="touch-icon-iphone.png"> <link rel="apple-touch-icon" sizes="76x76" href="touch-icon-ipad.png"> <link rel="apple-touch-icon" sizes="120x120" href="touch-icon-iphone-retina.png"> <link rel="apple-touch-icon" sizes="152x152" href="touch-icon-ipad-retina.png">
Internet Explorer и Windows Phone.

Домашний экран Windows 8 поддерживает четыре различные компоновки для постоянных размеров изображений и ожидает наличия четырех иконок. Можно не использовать те мета теги, которые не поддерживает ваше приложение.

<meta name="msapplication-square70x70logo" content="icon_smalltile.png"> <meta name="msapplication-square150x150logo" content="icon_mediumtile.png"> <meta name="msapplication-wide310x150logo" content="icon_widetile.png">

Цветные элементы браузера.

Chrome, Firefox OS, Safari, Internet Explorer и Opera Coast позволяют, используя различные мета теги, настроить элементы как браузера, так и платформы. Некоторые теги могут рабоать только для определенных платформ или браузеров.

Метатег цветовой темы для Chrome, Firefox OS и Opera <meta name="theme-color" content="#4285f4"> Метатег стартового изображения Safari. По умолчанию этот браузер пустой экран занрузки и после нескольких загрузок отображает скриншот предыдущего состояния приложения. Для явного указания стартового экрана применяется следующий тег <link rel="apple-touch-startup-image" href="icon.png"> Метатег изменения внешнего вида панели статуса устройства, в атрибуте contentиспользуется значение "black" вместо значения "black-translucent". <meta name="apple-mobile-web-app-status-bar-style" content="black">
